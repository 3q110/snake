# 萌萌贪吃蛇

可爱卡通风格的浏览器贪吃蛇小游戏（单文件 HTML）。

**版本：** `snake.1.2.0`

## 在线玩

https://3q110.github.io/snake/

## 本地运行

双击或用浏览器打开 `index.html` 即可。

## 功能

- 20×20 网格，经典撞墙模式
- 容易 / 中等 / 困难 三档难度
- 键盘与触屏操作
- **手感优化：方向缓冲**（快速连按/连滑更不容易吞转向）
- **道具系统**
  - ⚡ 加速：短时间更快
  - 🛡 护盾：碰撞时救命一次（取消本次移动）
  - ✨ 双倍分：短时间吃到食物得分翻倍
- **成就 & 皮肤**：达成目标解锁成就与新皮肤（在菜单/结算界面可查看与切换）
- **可爱动感音效**：道具拾取/到期、护盾救命、双倍分等都有专属反馈
- **声音控制**：HUD 右侧支持 🔊/🔇 静音与音量滑杆
- 分难度最高分记录（localStorage）

## 操作

- **方向键** 或 **WASD** 移动
- **空格** 暂停
- HUD：🔊/🔇 静音，滑杆调音量

## 开发 / 自测

- 打开 `index.html#test` 可运行内置自测用例（只做逻辑自检，不会启动游戏循环）
