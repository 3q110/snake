# 萌萌贪吃蛇

可爱卡通风格的贪吃蛇小游戏，提供 **H5 网页版** 和 **微信小程序版** 双端体验。

**版本：** `snake.1.2.1`

## 在线玩

- **网页版：** https://3q110.github.io/snake/
- **微信小程序：** 使用微信开发者工具打开 `miniprogram/` 目录（AppID: `wxca36be1a98b67bba`）

## 本地运行

### 网页版

双击或用浏览器打开 `index.html` 即可。

### 微信小程序版

1. 下载安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开工具，选择「导入项目」
3. 项目目录选择本仓库的根目录 `snake/`
4. AppID 填写：`wxca36be1a98b67bba`
5. 点击导入，即可预览

## 功能

- 30×30 网格（小程序版）/ 20×20 网格（网页版），经典撞墙模式
- 容易 / 中等 / 困难 三档难度
- 键盘 & 触屏 & D-pad 方向键操作
- **手感优化：方向缓冲**（快速连按/连滑更不容易吞转向）
- **道具系统**
  - ⚡ 加速：短时间更快
  - 🛡 护盾：碰撞时救命一次（取消本次移动）
  - ✨ 双倍分：短时间吃到食物得分翻倍
- **成就 & 皮肤**：达成目标解锁成就与新皮肤（在菜单/结算界面可查看与切换）
- **可爱动感音效**：道具拾取/到期、护盾救命、双倍分等都有专属反馈
- **声音控制**：HUD 右侧支持 🔊/🔇 静音与音量滑杆

## 操作

### 网页版

- **方向键** 或 **WASD** 移动
- **空格** 暂停
- 触屏滑动控制方向

### 小程序版

- 触屏滑动控制方向
- 屏幕下方 **D-pad** 方向按钮
- 暂停/继续通过 overlay 按钮

## 项目结构

```
snake/
  index.html              # H5 网页版（单文件）
  miniprogram/            # 微信小程序版
    app.js / app.json / app.wxss
    pages/game/           # 游戏页（单页即玩）
    utils/
      core.js             # 平台无关游戏核心逻辑
      storage.js           # wx storage 封装
      audio.js             # 音效池（InnerAudioContext）
    assets/audio/         # 软萌泡泡糖风格音效文件
  project.config.json     # 微信开发者工具配置
```
